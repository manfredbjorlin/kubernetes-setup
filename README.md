# k0s
Single-node
- install k0scli and run `k0scli apply -f 00-k0scli-configuration.yaml`
- run cilium download and install
- if `<pending>` EXTERNAL_IP run the cilium yaml - edit to add correct ips

# k3s
Single-node
- run both k3s scripts
- run cilium download and install scripts
- if `<pending>` EXTERNAL_IP run the cilium yaml - edit to add correct ips
